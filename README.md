# Spring boot demo for API to manage the CRUD of a ticket

API REST services defined using Swagger/OpenAPI

## Entorno, Herramientas y dependencias destacadas
	
	IntelliJ IDEA 2021.2
	
	Maven 3.3
	Spring Boot 2.5.3
	Java 8
	
	SpringBoot DevTools
	SpringBoot starter Data jpa
	H2 Database
	Lombok
	Mockito
	JUnit 5
	Springdoc OpenAPI
